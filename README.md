# Beon Technical Assessment

This repository was created for the BeOn technical assessment and contains the proposed solution, code organization, and required analyses according to the challenge specifications.

## Objective
Demonstrate analytical thinking, good data practices, and clear communication of results.

## Structure
- `data/` — Input and processed data
- `notebooks/` — Exploratory analysis and solution development
- `src/` — Reusable scripts and functions
- `results/` — Outputs, metrics, and final results

## How to Run
1. Clone the repository  
2. Install the required dependencies  
3. Run the notebooks or scripts as described in each folder

## Notes
This project was developed exclusively for evaluation purposes
